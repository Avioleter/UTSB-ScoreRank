# 北科大教务系统 - 成绩排名增强 v2.4

油猴脚本，自动在成绩页面的「备注」列显示排名徽章，支持导出 Excel，计算加权平均分。

## 功能

- 🔵 **自动排名显示** — 每门课的「备注」列自动显示排名标签（`排名 3/31`），前三名金色高亮
- 📊 **加权平均分** — 右下角面板实时显示加权平均分
- 📥 **导出 Excel** — 一键导出成绩数据（含排名、学分、课程性质等 12 列）
- 🔄 **自动刷新** — 切换学期 / 翻页时自动重新注入排名

## 安装

### 方法一：一键安装（推荐）

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 [ustb-grades-enhancer.user.js](ustb-grades-enhancer.user.js) → 点击右上角 **Raw**（或直接打开 raw 链接），Tampermonkey 会自动弹出安装提示
3. 点击 **安装** 即可

### 方法二：手动复制

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 新建脚本，复制 `ustb-grades-enhancer.user.js` 全部内容粘贴进去
3. `Ctrl+S` 保存

安装后打开 [成绩查询页面](https://byyt.ustb.edu.cn/cjgl/grcjcx/grcjcx)，等待约 1 秒即可看到排名。

## 更新日志

### v3.0.0 — 面板式成绩增强（[ustb-grades-enhancer-v3.user.js](ustb-grades-enhancer-v3.user.js)）

v3.0 为全新版本，采用独立面板展示，与 v2.4 并行维护，推荐使用。

**下载：**[ustb-grades-enhancer-v3.user.js](ustb-grades-enhancer-v3.user.js)

**新增功能：**

- 🔍 **平时/考试成绩** — 调用教务系统 `seeFx` API 获取每门课的分项成绩（平时分、考试分、实验分），4 并发查询
- 🪟 **独立面板** — 不再修改教务页面 DOM，所有排名和分项成绩在自有面板中展示，零冲突
- 📊 **智能加权计算** — 只计入必修课、实验课、平台课，自动排除素质拓展、专业拓展、公共选修等非核心课程
- 📥 **Excel 导出增强** — 新增平时分、考试分、实验分列，共 15 列

**使用方式：**

1. 打开成绩页面 → 右下角出现 📊 按钮
2. 点击 📊 → 弹出面板，数据自动加载
3. 分项成绩自动获取（或点击面板中「获取平时/考试成绩」手动触发）
4. 按 `Esc` 或点遮罩关闭面板

**调试命令（控制台）：**

```javascript
__USTB__.dump()     // 查看课程摘要表（含平时分/考试分）
__USTB__.raw(0)     // 查看第一门课的原始分项数据
__USTB__.detail()   // 手动触发分项成绩获取
```
